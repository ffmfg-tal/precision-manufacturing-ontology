# Reference Implementations

This directory holds mappings from the canonical ontology (see `../reference/domain-map.md`, `../extensions/`, `../schemas/`) to specific systems. These are *reference implementations* — examples of how a conforming system might realize the ontology. They are not the canon.

## Current mappings

- `carbon.md` — Carbon OS (https://github.com/crbnos/carbon)
- `fulcrum.md` — Fulcrum Pro (the manufacturer's current system of record)
- `_template.md` — starting point for documenting a new implementation
